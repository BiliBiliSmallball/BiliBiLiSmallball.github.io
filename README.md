## 项目架构

bilibilismallball.github.io/
├── _config.yml          # Jekyll 配置文件
├── _posts/              # 博客文章目录
│   ├── 2024-02-16-first-post.md
│   └── 2024-02-17-second-post.md
├── _layouts/            # 布局文件
│   ├── default.html     # 默认布局
│   └── post.html        # 文章布局
├── _includes/           # 可重用组件
│   ├── header.html      # 头部
│   ├── footer.html      # 页脚
│   └── nav.html         # 导航栏
├── assets/              # 静态资源
│   ├── css/             # CSS 文件
│   ├── js/              # JavaScript 文件
│   └── images/          # 图片文件
├── index.html           # 首页
├── about.md             # 关于页面
├── contact.md           # 联系页面
└── _site/               # Jekyll 生成的静态文件（自动生成）
